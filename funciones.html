Funciones PHP

Funcion en PHP

las funciones son bloques de codigo donde podemos definir X funcionalidad dentro de nuestro programa. la cual cumple cierta funcionalidad dentro de nuestro scope de esta manera encapsulamos nuestra logica dentro de nuestro programa para poder ser reutilizados para definir una funcion necesitamos utilizar la palabra resevada function seguida por le nombre del mismo. las funciones dentro de PHP OOP o programacion orientado a Objetos se les conoce como Metodos ya que estos dependen de una clase para saber mas de ellos te invito a leer la documentacion de PHP. en este documento lo analizaremos despues en la seccion de OOP PHP.

Ejemplo de como escribir una funcion

function sumar() {
 $numero1 = 1;
 $numero1 = 2;
 $suma = $numero1 + $numero1;

 echo $suma;
}

sumar();

Ahora llamamos a nuestra funcion para poder utilizarla

Funciones con Argumentos

las funciones conn argumentos son aquella a las que se le pasa cualquier tipo de informacion. la cual es una lista de expresiones delimitadas por comas. Los argumentos son evaluados de izquierda a derecha.
function funcion_con_argumetos($valor) {
  echo $valor;
}

Mandamos llamar nuestra funcion

funcion_con_argumetos();

Funciones que retornan un valor

las funciones que retornan un valor con la palabra reservada return. en PHP 7 las funciones o metodos pueden ser asignadas el tipo de dato que deseas que sea retornado. ya sea un entero, un string o un booleano etc. segun lo requieras pero eso lo veremos en la seccion de PHP 7
function funcion_con_argumetos($valor) {
  return $valor;
}

Mandamos llamar nuestra funcion

funcion_con_argumetos();
